# Как развернуть сайт на Render (получить ссылку)

Render — облачный хостинг с бесплатным планом. Ссылка будет вида:
**https://ecp-registry-romanenko.onrender.com**

Работает 24/7. Преподаватель открывает в браузере с любого устройства.

⚠️ Важно: Render усыпляет бесплатные сервисы через 15 минут бездействия.
При первом обращении после паузы сайт «просыпается» 30-50 секунд.
Перед защитой просто открой сайт за минуту до показа — он будет уже активен.

---

## Что понадобится

1. Аккаунт **GitHub** (если нет — зарегистрируйся: https://github.com)
2. Аккаунт **Render** (https://render.com — бесплатно)

Можно регистрироваться через Google — займёт 2 минуты на каждый сервис.

---

## Шаг 1. Создаём репозиторий на GitHub (5 минут)

### Вариант А: через веб-интерфейс GitHub (проще)

1. Зайди на https://github.com → войди
2. В правом верхнем углу нажми **"+"** → **"New repository"**
3. Параметры:
   - **Repository name**: `ecp-registry`
   - Видимость: **Public** (для бесплатного Render это обязательно)
   - НЕ ставь галочки на "Add README", "gitignore", "license"
4. Нажми **"Create repository"**
5. На открывшейся странице найди раздел **"...or upload an existing file"** и нажми его

### Загружаем файлы

6. Распакуй на своём компьютере архив `site_ecp_registry.zip`
7. Внутри распакованной папки `site` выдели **все файлы и папки** (Ctrl+A)
8. Перетащи их прямо в окно GitHub (туда где написано "Drag files here")
9. Внизу страницы введи commit message: `Initial commit`
10. Нажми **"Commit changes"**

Готово — твой код на GitHub.

---

## Шаг 2. Подключаем Render (5 минут)

1. Зайди на https://render.com → нажми **"Get Started for Free"**
2. Зарегистрируйся через GitHub (выбери опцию **"GitHub"**) — это автоматически даст Render доступ к твоему репозиторию
3. После регистрации на главной панели нажми **"+ New"** → выбери **"Web Service"**
4. В списке репозиториев найди **`ecp-registry`** и нажми **"Connect"**

### Настройка сервиса

Заполни форму:

- **Name**: `ecp-registry-romanenko` (или любое — это войдёт в ссылку: `https://ecp-registry-romanenko.onrender.com`)
- **Region**: Frankfurt (ближе всех к Калининграду)
- **Branch**: `main`
- **Runtime**: Python 3
- **Build Command**: `pip install -r requirements.txt && python init_db.py`
- **Start Command**: `gunicorn app:app`
- **Instance Type**: **Free** (внизу)

5. Нажми **"Create Web Service"**

## Если что-то не работает

### Сайт не открывается / Internal Server Error

Зайди на Render → твой сервис → вкладка **"Logs"**. Там увидишь ошибку.
Пришли мне скриншот — поправим.

### Сайт «висит» при открытии

Это нормально, если он спал больше 15 минут. Подожди 30-50 секунд — проснётся.

### Хочу сменить адрес ссылки

Render → твой сервис → Settings → Name. Меняешь — сайт получает новую ссылку.
Старая перестаёт работать сразу.

### Хочу убрать "засыпание"

Это особенность бесплатного плана. Платный план $7/месяц убирает засыпание.
На защиту хватит и бесплатного — просто открой сайт за минуту до показа.
