Есть три варианта по сложности — выбери любой.

================================================================
ВАРИАНТ 1 — PythonAnywhere (постоянная ссылка, бесплатно, 15 минут)
================================================================

Получишь ссылку вида: https://romanenko.pythonanywhere.com
Работает 24/7, преподаватель открывает в браузере с любого устройства.

## Шаг 1. Регистрация (2 минуты)

1. Зайди на https://www.pythonanywhere.com
2. Нажми "Pricing & signup" → выбери тариф **"Beginner — Free"**
3. Username придумай аккуратно — он войдёт в адрес сайта. Например, `romanenko-ecp` → 
   сайт будет `romanenko-ecp.pythonanywhere.com`
4. Подтверди email

## Шаг 2. Загружаем файлы (5 минут)

1. На главной панели нажми **"Files"** (вкладка вверху)
2. В корневой папке создай новую папку: **"Add a new directory"** → название `ecp_registry`
3. Зайди в эту папку
4. Загрузи туда содержимое архива `site_ecp_registry.zip`:
   - распакуй архив на своём компьютере
   - в PythonAnywhere в этой папке нажми **"Upload a file"** и загрузи по одному:
     - `app.py`
     - `init_db.py`
     - `schema.sql`
     - `requirements.txt`
     - `README.md`
5. Создай подпапки `static`, `templates`, `instance` и в них загрузи соответствующие файлы
   (либо проще: воспользуйся консолью — см. ниже)

### Альтернатива через консоль (быстрее):

1. На главной нажми **"Consoles"** → **"Bash"**
2. В терминале выполни:
   ```bash
   cd ~
   mkdir ecp_registry
   cd ecp_registry
   ```
3. Затем вернись в Files и загрузи `site_ecp_registry.zip` в свою домашнюю папку.
4. В консоли:
   ```bash
   cd ~/ecp_registry
   unzip ~/site_ecp_registry.zip
   mv site/* .
   rmdir site
   ```

## Шаг 3. Инициализируем БД (1 минута)

В консоли Bash:
```bash
cd ~/ecp_registry
python3 init_db.py
```

Должно вывестись:
```
Сотрудников: 52 (активных: 45)
Токенов: 25
Сертификатов: 49
МЧД: 26
Пользователей: 3
Инициализация БД завершена.
```

## Шаг 4. Настраиваем веб-приложение (5 минут)

1. Вверху страницы PythonAnywhere → **"Web"**
2. Нажми **"Add a new web app"** → Next → выбери **"Manual configuration"** → **Python 3.10**
3. Откроется страница настроек твоего сайта. Найди раздел **"Code"**:
   - **Source code**: введи `/home/ТВОЁ_USERNAME/ecp_registry`
   - **Working directory**: то же самое `/home/ТВОЁ_USERNAME/ecp_registry`
4. Найди раздел **"WSGI configuration file"** — нажми на ссылку (откроется редактор).
   Удали всё содержимое и вставь:
   ```python
   import sys
   import os
   
   project_home = '/home/ТВОЁ_USERNAME/ecp_registry'  # ЗАМЕНИ username!
   if project_home not in sys.path:
       sys.path.insert(0, project_home)
   
   os.chdir(project_home)
   
   from app import app as application
   ```
   Замени `ТВОЁ_USERNAME` на свой логин PythonAnywhere. Сохрани (Ctrl+S).
5. Вернись на вкладку Web и нажми зелёную кнопку **"Reload"** вверху.

================================================================
ВАРИАНТ 2 — ngrok (быстрая ссылка прямо с твоего ноутбука, 5 минут)
================================================================

Подходит когда нужно показать сайт **прямо сейчас** во время защиты.
Ссылка живёт пока твой ноутбук включён.

## Шаг 1. Установи Python и Flask (если ещё нет)

Открой командную строку (cmd на Windows, Terminal на Mac):
```
pip install flask werkzeug
```

## Шаг 2. Запусти сайт

1. Распакуй `site_ecp_registry.zip` в любую папку
2. В командной строке зайди в эту папку:
   ```
   cd путь/к/папке/site
   python init_db.py
   python app.py
   ```
3. Должно появиться: `Running on http://127.0.0.1:5000`
4. Открой в браузере http://127.0.0.1:5000 — должен работать локально

## Шаг 3. Поднимаем туннель ngrok

1. Зарегистрируйся на https://ngrok.com (1 минута)
2. Скачай ngrok для своей ОС (Windows/Mac)
3. Распакуй ngrok в любую папку
4. В личном кабинете ngrok скопируй authtoken и выполни:
   ```
   ngrok config add-authtoken ТВОЙ_ТОКЕН
   ```
5. Запусти туннель (в новом окне командной строки, не закрывая Flask):
   ```
   ngrok http 5000
   ```
6. Появится ссылка вида:
   ```
   Forwarding   https://abc123.ngrok-free.app -> http://localhost:5000
   ```

Эту ссылку https://abc123.ngrok-free.app отдаёшь преподавателю — он откроет сайт
который физически работает у тебя на ноутбуке.

**Минусы ngrok:**
- Ссылка работает только пока твой ноутбук включён и ngrok запущен
- Бесплатная ссылка меняется при каждом перезапуске ngrok

================================================================
ВАРИАНТ 3 — На компьютере филиала
================================================================

Самый сильный вариант для защиты, но требует доступа к серверу.
1. Распаковать `site_ecp_registry.zip` на компьютере филиала
2. Установить Python и Flask  
3. Запустить `python app.py` — будет работать в локальной сети филиала
4. Преподаватель открывает http://192.168.x.x:5000 (адрес скажет сисадмин)
